2) The Four Hyperlink Types

1. External Link

<a href="https://www.w3.org"
   target="_blank"
   rel="noopener">
  Visit W3C
</a>
Navigates to another website in a new tab. rel="noopener" prevents security vulnerabilities.

2. Internal Link

<a href="about.html">
  About Page
</a>
Navigates to another page within the same website. Use relative paths.

3. Anchor Link (Jump Link)

<!-- Target element: -->
<h2 id="section-2">Section 2</h2>
<!-- Link to it: -->
<a href="#section-2">Go to Section 2</a>
Jumps to an element with a matching id on the same page. Used in table of contents.

4. Email & Tel Links

<a href="mailto:info@example.com">
  Email us
</a>
<a href="tel:+15551234567">
  Call us
</a>
Opens default email client or phone dialer. mailto: and tel: are URL schemes.

5) Image as a Hyperlink

Wrapping an <img> inside an <a> makes the image clickable.

<a href="https://www.w3.org" target="_blank">
  <img src="w3c-logo.png"
       alt="Visit W3C website"
       width="120" />
</a>
Result (simulated):
W3C
alt: Visit W3C website

The entire image is clickable. Always include alt text describing the link destination.
